# Navigation Improvements

## Remove Current Structure

Current navigation hierarchy is confusing because it mixes:

* Destinations
* Tour Packages
* Nile Cruises

within the same menu paths.

Example:

```
Luxor
  └── Egypt Packages
       └── Nile Cruises
```

This does not match how travelers search for trips.

---

## New Navigation Logic

### Tours

For visitors searching by experience type.

Examples:

* Classic Egypt Tours
* Luxury Egypt Tours
* Family Tours
* Private Tours

### Nile Cruises

For visitors specifically interested in cruising.

Examples:

* 3 Night Cruises
* 4 Night Cruises
* Luxury Cruises
* Dahabiya Cruises

### Destinations

For visitors researching locations.

Examples:

* Cairo
* Luxor
* Aswan
* Alexandria
