Access Egypt Travel — SEO Plan

Step-by-step instructions for Rank Math setup, keyword targeting, and content pages. Start AFTER the developer finishes Phase 1.

Plugin: Rank Math SEO (already installed) Site: accessegypttravel.com Status: Phase 2 — after developer Phase 1 Prepared: June 2026
⚠️ Do NOT start these tasks until the developer completes developer-plan.html first. SEO on broken/demo content is wasted effort — clean the site first, then optimize.

Why SEO Is Phase 2 (Not Phase 1)

The developer's job (Phase 1) is to clean up the site: delete 18 Lorem Ipsum blog posts, remove 14 junk pages, fix broken elements, and rebuild the footer and contact page. Until that's done, doing keyword work is pointless — Google is already penalizing the domain because of those junk posts, and any meta descriptions you set on the homepage get undermined by 18 fake blog posts about "Bookmarksgrove" also indexed on the same domain.

The right order: Developer finishes Phase 1 → You do Step 1 (sitemap) → Then do the rest of this document in order.

What's already installed: Rank Math SEO is active on the site. You just haven't configured it yet. Every step below is you using Rank Math — no new plugins needed.

Do First

Step 1 — Submit Updated Sitemap to Google Search Console

Why: After the developer deletes all junk pages and blog posts, Rank Math automatically updates the sitemap. Submitting it tells Google "please re-crawl — things have changed." Without this, deleted pages can stay in Google's results for weeks.
1
Go to Google Search Console
https://search.google.com/search-console

Log in with the Gmail account connected to your website (likely accessegypttravel@gmail.com). If you haven't verified the site yet, you'll need to do that first — Rank Math has a one-click Google Search Console connection under Rank Math → General Settings → Webmaster Tools → Google Search Console.

2
Navigate to Sitemaps
Left sidebar → Sitemaps
3
Submit the Sitemap URL

In the "Add a new sitemap" field, type this URL exactly:

Copy and paste this URL
https://accessegypttravel.com/sitemap_index.xml

Click Submit. Google will confirm it was received. It may take 1–3 days for Google to fully re-crawl.

4
Request Removal of Specific Junk Pages
Why: The sitemap submission speeds up crawling, but doesn't immediately remove already-indexed pages. The Removals tool does.
Left sidebar → Removals → New Request

Add these URLs one by one — they are the worst junk pages most likely to have been indexed:

Paste each of these as a separate Removal Request
https://accessegypttravel.com/george https://accessegypttravel.com/tours-2 https://accessegypttravel.com/?p=1 (and any other ?p= numbered URLs) Any /blog/ post URLs with "lorem" or placeholder titles

For each one: paste the URL → click Continue → select "Remove this URL only" → click Submit Request.

Phase 2 — Rank Math

Step 2 — Set Focus Keywords on Every Tour Page

Why: Without a focus keyword, Rank Math gives every page a red/orange score and Google has no clear signal for what each page should rank for. This is the single biggest Rank Math quick win — takes about 30 minutes for the whole site.

How to Set a Focus Keyword (do this for each page below)

1
Open the page in the WordPress editor
Pages → All Pages → hover the page name → click Edit
Do NOT use Quick Edit — you need to be inside the full editor to see the Rank Math panel.
2
Scroll down to the Rank Math SEO panel

Scroll past the Elementor editor area to the bottom of the page. You'll see a panel called "Rank Math SEO" — it has a score (currently probably red/orange).

If you don't see it, click the three dots (...) at the top right of the editor → Preferences → make sure "Rank Math SEO" is toggled on.
3
Click "Focus Keyword" and type your keyword

Inside the Rank Math panel, find the Focus Keyword field. Click it and type (or copy from the table below). Press Enter after typing — Rank Math will run its analysis.

Rank Math SEO panel → Focus Keyword field → type keyword → press Enter
4
Click "Update" to save the page
Top right corner → click blue Update button

Phase 2 — Rank Math

Step 3 — Write Meta Descriptions via Rank Math

Why: The meta description is the 1–2 lines of text that appear under your page title in Google search results. It doesn't directly affect your ranking, but it's the first thing a searcher reads before deciding whether to click. A good meta description gets you clicked; a missing or generic one gets you ignored.

How to Edit a Meta Description (do this for each page below)

1
Open the page editor (same way as Step 2)
Pages → All Pages → hover page name → Edit
2
Open the Rank Math "Edit Snippet" panel
Scroll down to Rank Math SEO panel → click Edit Snippet button (it looks like a Google search result preview)

A preview box opens showing how your page looks in Google search results. You'll see editable fields for Title and Description.

3
Click the Description field and paste your text

Click inside the Description field. Select all existing text (Ctrl+A or Cmd+A) and delete it. Then paste the description from the table below. Keep it under 155 characters — Rank Math will show a warning if it's too long.

4
Click "Update" to save
Top right → Update

Step 5 — Add Pickup Location Mentions Across the Site

Why this is your biggest SEO opportunity: American tourists at Red Sea resorts (Hurghada, Sharm el-Sheikh) regularly search "Hurghada to Luxor day trip" or "day tour from Hurghada." This is a high-intent search — they want to book something. Access Egypt Travel offers exactly this, but the words "Hurghada to Luxor" appear almost nowhere on the site right now. Same for "Cairo to Luxor." Both are easy wins.

Add the pickup location text to each of these places — the developer can do this, or you can do it yourself in Elementor:

Where on the SiteWhat to AddExact Text to Use
Homepage — Hero section subtitle One line below the main headline
Pickup available from Cairo, Hurghada & Luxor
Each tour card on tour listing pages A small "Departs from:" line under the tour title
📍 Departs from: Hurghada or Cairo
Each individual tour page — first paragraph Mention pickup in the opening description
This tour is available with hotel pickup from Cairo or Hurghada, with transfer to Luxor included.
Contact page — Address/Location section Replace single "Hurghada" address with multi-city
