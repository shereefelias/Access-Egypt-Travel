Schema Markup — Copy-Paste JSON-LD Reference

Ready-to-use structured data for accessegypttravel.com. Customize the placeholder values marked with [BRACKETS], then add to WordPress via Rank Math or in the site's <head>.

⚡ Priority order: FAQPage first (fastest rich result), then Organization, then LocalBusiness. Tour pages and content pages can be done last. Test each one at search.google.com/test/rich-results after adding.

How to Add Schema Markup in WordPress

Two methods — use Rank Math (preferred) or paste directly into the page header

Method 1: Rank Math (Recommended)

Edit any page → Rank Math sidebar panel → Schema tab → + Add Schema → choose schema type → fill in the form. Rank Math outputs clean JSON-LD automatically.

For schema types not available in Rank Math's dropdown (like the full Organization schema below), use Method 2.

Method 2: WordPress Custom Header Code

Rank Math → General Settings → Others → Header Code — paste JSON-LD directly. This adds the code site-wide (appropriate for Organization and LocalBusiness). For page-specific schema, use a plugin like Insert Headers and Footers to add code per-page.

Always validate schema at search.google.com/test/rich-results after adding. Paste your page URL and it will show what Google can read and any errors.

Priority 1 FAQPage Schema — FAQ Page

Produces FAQ accordion in Google results. Fastest visible win — can appear in 24–48 hours.

Where to add: FAQ page only (accessegypttravel.com/faq/ or similar). Edit the FAQ page → Rank Math → Schema → + Add Schema → FAQ. Or paste the code below via Insert Headers and Footers on the FAQ page only.

What to customize

  • Replace each [QUESTION] with the exact question text as it appears on your FAQ page
  • Replace each [ANSWER] with a concise 1–3 sentence answer. Google truncates long answers.
  • Add or remove mainEntity entries to match the number of FAQs on your page
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "Is Egypt safe for American tourists?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Yes. Major tourist areas — Cairo, Luxor, Aswan, and Hurghada — are safe for American visitors. The U.S. State Department rates Egypt Level 2 (Exercise Increased Caution), the same rating as France and Germany. Our guides have operated tours for American clients in these areas for over 22 years without a safety incident."
      }
    },
    {
      "@type": "Question",
      "name": "Do Americans need a visa to visit Egypt?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Yes. American tourists require a visa to enter Egypt. The easiest option is the Egypt e-Visa, available online at visa2egypt.gov.eg for approximately $25 USD. Processing takes 3–5 business days. Visa on arrival is also available at Cairo, Hurghada, and Luxor airports for $25 USD cash."
      }
    },
    {
      "@type": "Question",
      "name": "What is the best time to visit Egypt?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "October through April offers the most comfortable temperatures for visiting Egypt's outdoor historical sites. October and November are especially recommended — warm but not extreme, with smaller crowds than the December-January peak season. Summer (June–August) is very hot in Upper Egypt but offers the lowest prices and fewest crowds."
      }
    },
    {
      "@type": "Question",
      "name": "Can I be picked up from Hurghada for a Luxor tour?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Yes. We offer hotel pickup from Hurghada for Luxor day tours and multi-day packages. The drive through the Eastern Desert takes approximately 3 hours. We also pick up from Cairo and Luxor itself."
      }
    },
    {
      "@type": "Question",
      "name": "How much does a private Egypt tour cost?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Private Egypt tour prices depend on duration, group size, and inclusions. We offer private day tours from Hurghada to Luxor, as well as multi-day packages covering Cairo, Luxor, Aswan, and the Nile. All prices are quoted in USD with no hidden fees. Contact us for a custom quote."
      }
    },
    {
      "@type": "Question",
      "name": "[YOUR NEXT FAQ QUESTION HERE]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "[YOUR ANSWER — 1 to 3 sentences]"
      }
    }
  ]
}
</script>
Each question in the schema must match a question that is visible on the page. Google cross-references the schema with the actual page content. If questions in the schema don't appear on the page, Google will ignore the schema.

Priority 4 Product Schema — Each Tour Page

Adds pricing and availability signals to tour pages. Repeat for each tour page, adjusting the values.

Where to add: Each tour page individually. Edit the tour page → Rank Math → Schema → + Add Schema → Product. Or use Insert Headers and Footers with page-level targeting.

What to customize per tour page

  • Change "name" to the exact tour name
  • Change "description" to match the tour's actual content
  • Change "price" to the tour's actual USD price — only if you're comfortable showing it in Google results. Use the starting price and note "from" in the description.
  • Change "image" to the featured image URL for that tour page
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "[TOUR NAME — e.g. 'Private Luxor Day Tour from Hurghada']",
  "image": "[URL OF TOUR PAGE FEATURED IMAGE]",
  "description": "[TOUR DESCRIPTION — 1-2 sentences: what's included, pickup locations, highlights]",
  "brand": {
    "@type": "Brand",
    "name": "Access Egypt Travel"
  },
  "offers": {
    "@type": "Offer",
    "url": "[URL OF THIS TOUR PAGE]",
    "priceCurrency": "USD",
    "price": "[STARTING PRICE IN USD — numbers only, e.g. 195]",
    "availability": "https://schema.org/InStock",
    "seller": {
      "@type": "Organization",
      "name": "Access Egypt Travel"
    }
  },
  "provider": {
    "@type": "Organization",
    "name": "Access Egypt Travel",
    "url": "https://accessegypttravel.com"
  }
}
</script>

Priority 6 Article Schema — Each of the 6 Content Pages

Marks content pages as editorial articles, signals authorship and publish date. Repeat for each content page.

Where to add: Each of the 6 content pages (visa, safety, best time, Hurghada-Luxor, packing, Cairo-Luxor). Edit each page → Rank Math → Schema → + Add Schema → Article.

What to customize per content page

  • Change "headline" to the H1 of the specific article
  • Change "url" to the actual URL of the specific page
  • Change "image" to the featured image URL for that page
  • Change "datePublished" and "dateModified" to match when the page was published/updated
  • Replace [JOHN_PHOTO_URL] with John's headshot URL
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "[H1 OF THIS PAGE — e.g. 'Is Egypt Safe for American Tourists?']",
  "url": "[FULL URL OF THIS PAGE]",
  "image": "[FEATURED IMAGE URL FOR THIS PAGE]",
  "datePublished": "2026-06-01",
  "dateModified": "2026-06-01",
  "author": {
    "@type": "Person",
    "name": "John Gamil",
    "url": "https://accessegypttravel.com/about-us/",
    "image": "[JOHN_PHOTO_URL]",
    "jobTitle": "Certified Egyptologist & Private Tour Guide"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Access Egypt Travel",
    "url": "https://accessegypttravel.com",
    "logo": {
      "@type": "ImageObject",
      "url": "[LOGO_URL]"
    }
  },
  "description": "[1–2 sentence summary of the article — same as the meta description]",
  "mainEntityOfPage": {
    "@type": "WebPage",
    "@id": "[FULL URL OF THIS PAGE]"
  }
}
</script>

Validate All Schema — Required Step

Don't skip this — errors in schema prevent rich results even if the code looks correct.

How to Test

After adding each schema block, go to:

https://search.google.com/test/rich-results

Paste the page URL and click Test URL. You'll see:

  • Green checkmark — schema is valid, page is eligible for this rich result type
  • Warning triangle — schema has errors, won't produce rich results until fixed
  • Red X — critical error in the JSON structure
Common errors: missing required fields (especially image on Article schema), mismatched question text between FAQPage schema and page content, and malformed URLs. The tool tells you exactly what's wrong.
