# Site Pop-up Photos — Drop-in Checklist

The hover pop-ups now show a **real photo** if one exists, and fall back to the colored gradient if it doesn't. To add photos:

1. Save a landscape JPG (roughly 512×200px or larger, ~16:6 crop looks best) for each site below.
2. Use the **exact filename** in the table (lowercase, matches the site key).
3. Put them all in `mockup/images/sites/`.
4. That's it — they appear automatically on hover. No code changes needed.

Until a file exists, that pop-up keeps showing its gradient, so partial coverage is fine.
