# Location Icons — Download Checklist (Flaticon Premium SVG)

How to use this:
1. On Flaticon, find **one Egypt icon pack by a single author** (search "Egypt" → open a pack you like → check it has most sites below). Using one pack keeps the style consistent.
2. Download each as **SVG** (Premium = no attribution needed).
3. Save into `mockup/icons/` using the **exact "Save as" filename** in the last column.
4. Tell me when they're in the folder — I'll wire them into all five tour pages and update the CSS so they render at the right size and scale crisply.

If the pack is missing a site, use the suggested search term to grab the closest match **in a similar style** (or skip it and I'll leave the current emoji as a fallback).
